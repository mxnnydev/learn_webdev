3 Types of Lists in HTML

Unordered List (<ul>)

Used for a list of items where the order doesn't matter.

  • Bread
  • Rice
  • Noodles

Ordered List (<ol>)

Used for steps or ranked items where the order matters.

  1. Wake up
  2. Brush teeth
  3. Eat breakfast

Description List (<dl>)

Used to define terms and their descriptions.

Ice Cream
A sweet, cold, creamy dessert.
Pizza
A cheesy dish originally from Italy.
Biryani
A spicy Indian rice dish made with meat or vegetables.
